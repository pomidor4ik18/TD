def initialize_game(): ¶

Инициализирует игровую среду для Tower Defence.

Осуществляет следующие действия: 1. Инициализирует Pygame и устанавливает окно игры. 2. Создает игровой мир на основе данных уровня. 3. Загружает карту, обрабатывает данные уровня и врагов. 4. Инициализирует группы спрайтов для турелей и врагов.

Глобальные переменные: - world (World): Объект игрового мира. - screen (pygame.Surface): Экран для отображения игры. - turret_group (pygame.sprite.Group): Группа спрайтов турелей. - enemy_group (pygame.sprite.Group): Группа спрайтов врагов. - clock (pygame.time.Clock): Объект для управления временем в игре.

Raises:
FileNotFoundError: Если отсутствует файл уровня ('levels/level.tmj') или карта ('levels/level.png').
